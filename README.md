es6-backbone-events
---

A no-frills es6 module for [Backbone](https://github.com/jashkenas/backbone)'s events.

## Dependencies:
  * underscore or comparable library aliased as 'underscore', installed separately

## Build Dependencies:
  * GNU sed, accessible as 'gsed'
